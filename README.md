# Monster Gaming Bug Tracker

Public issue tracker for the [Monster Gaming](https://monstergaming.ai) platform.

## How to Report a Bug

1. Check [existing issues](https://github.com/Excrothe/monstergaming-bugs/issues) to avoid duplicates
2. Click **New Issue** and select **Bug Report** or **Feature Request**
3. Fill out the template completely

## What Belongs Here

- Bug reports for the Monster Gaming platform
- Feature requests
- Integration issues (Unreal, Unity, Godot)

## What Does NOT Belong Here

- Security vulnerabilities (email security@luxedeum.com instead)
- Support questions (use the platform's built-in support)
- Issues with third-party tools not related to Monster Gaming

## Labels

| Label | Meaning |
|-------|---------|
| `bug` | Confirmed bug |
| `triage` | Needs review |
| `enhancement` | Feature request |
| `critical` | Crash or data loss |
| `engine:unreal` | Unreal Engine specific |
| `engine:unity` | Unity specific |
| `engine:godot` | Godot specific |
| `api` | API / Gateway issue |
| `wontfix` | Won't be addressed |
